# Store Listing — GSC Bulk Indexer

## Name

```
GSC Bulk Indexer
```

## Short description

```
Batch-submit URLs to Google Search Console for indexing. Auto-skips already-indexed URLs (3s check), detects daily quota limits, dismisses dialogs automatically. Save hours of manual work.
```

## Detailed description

```
GSC Bulk Indexer automates submitting multiple URLs through the Google Search Console URL Inspection tool for indexing requests. Built for SEO professionals and site owners who need to reindex content at scale.

HOW IT WORKS
1. Paste URLs or a sitemap.xml into the popup.
2. Open your Search Console URL Inspection page.
3. Click RUN — it types each URL, checks if already indexed (skips in ~3s), clicks Request Indexing, and dismisses the dialog.

FEATURES
- Auto-skip already-indexed URLs (3-second check per URL)
- Sitemap.xml support — paste the entire XML, URLs extracted automatically
- Quota detection — stops when your daily Request Indexing limit is hit
- Auto-dismiss confirmation dialogs immediately
- Resume support — remembers completed URLs if stopped mid-run
- Real-time progress bar, per-URL status, and full activity log
- Dark retro synthwave UI with monospace fonts and scanlines
- No analytics, no servers, no third-party code

USE CASES
- Bulk reindexing after website migration or domain change
- Submit newly published content for faster discovery
- Recover from soft 404s or URL structure changes
- Agency workflow — manage indexing across multiple properties

Developer: Nasratul nayem — https://github.com/nasratulnayem
```
